
# 🇺🇸 Análisis y Clasificación de Presidentes de EE. UU. en Google Colab

Este notebook ha sido adaptado para ejecutarse fácilmente en Google Colab, permitiéndote analizar datos históricos sobre los presidentes de Estados Unidos y aplicar técnicas de Machine Learning para clasificar la duración de sus mandatos.

## 📥 Cómo abrir en Google Colab

1. Ve a 👉 [https://colab.research.google.com](https://colab.research.google.com)
2. Haz clic en **“Archivo” > “Subir notebook”**
3. Selecciona el archivo `ProyectoDS_Parte_I_Fernandez_final_COLAB.ipynb` que descargaste
4. Ejecuta las celdas una por una (Shift + Enter)

## 📌 Requisitos

Google Colab ya incluye todas las librerías necesarias (`pandas`, `scikit-learn`, etc.). No necesitas instalar nada adicional.

## 🔍 ¿Qué incluye el notebook?

- Limpieza y preparación del dataset `us_presidents 2.csv`
- Cálculo de la duración de cada mandato presidencial
- Creación de una variable categórica `mandate_class`
- Codificación de variables categóricas
- Feature Selection con `SelectKBest`
- Modelo de clasificación con `RandomForestClassifier`
- Evaluación del modelo con métricas y matriz de confusión

## 📊 ¿Qué se puede aprender?

Este proyecto te enseña a:
- Convertir variables históricas en datos numéricos útiles para modelos predictivos
- Aplicar clasificación supervisada a datos no convencionales
- Evaluar un modelo y reflexionar sobre sus limitaciones y mejoras
